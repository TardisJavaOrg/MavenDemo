# Java Web 基础，看这一个项目就够了

本项目通过几个项目，让你对 Servlet 到 Spring Cloud 到开发有一个了解。进行最快速的入门，且抛弃无用的多余配置文件。

能通过注解就通过注解进行编写。

因为现在开发早已经不再使用原生 Servlet 进行开发，微服务分布式早已占领高低。

但是无论他们如何占领高低，地基还是 Servlet，所以了解Servlet，能对学习新技术有一个理论基础。

而不是本地起高楼，没有地基，理解会很吃力。

> 学得越多，学得越快。 -- 鲁迅
